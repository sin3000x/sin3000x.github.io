``` 
   / \          / \
  2   3   ->   5   2
 / \              / \
4   5            3   1
```
单元:
```
  A          B
 / \   ->   / \
B   C      C   A
```
```py
root.left.left = root.right
root.left.right = root
root.left = None
root.right = None
```

```py
def upside(root):
    if root is None:
        return root
    if root.left is None and root.right is None:
        return root
    new_root = upside(root.left)
    root.left.left = root.right
    root.left.right = root
    root.left = None
    root.right = None
    return new_root
```
时间: $O(n/3)=O(n)$

空间: $O(n/3)=O(n)$
