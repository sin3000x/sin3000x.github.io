`"aabbbccd"` 返回 `"abcd"`.

利用快慢指针, `fast` 负责遍历, `slow` 左边的是要保留的, 没有重复的字符. 可以做到原地修改.
```py
def remove_duplicate(string):
    if len(string) in (0, 1):
        return string

    string = list(string)
    slow, fast = 1, 1
    while fast < len(string):
        if string[fast] != string[slow-1]:
            string[slow] = string[fast]
            slow += 1
        fast += 1
    return ''.join(string[:slow])
```
