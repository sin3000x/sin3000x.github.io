比如 `8 -> 5 -> 8 -> 8 -> 1 -> 8` 删除 `8`, 返回 `5 -> 1`.

```py
def remove(head, val):
    if head is None:
        return head

    fake = ListNode(None)
    fake.next = head
    prev, cur = fake, head

    while cur:
        if cur.val == val:
            prev.next = cur.next
        else:
            prev = cur    
        cur = cur.next

    return fake.next
```
