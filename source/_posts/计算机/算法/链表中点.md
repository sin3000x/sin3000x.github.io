用快慢指针的方法找链表的中点.

<!--more-->

## 方法一
```python
while fast and fast.next:
    fast = fast.next.next
    slow = slow.next
```
结果:
```
a -> b -> c -> d -> e
          S         F
```
```
a -> b -> c -> d -> e -> f
               S            F
```

## 方法二
```python
while fast and fast.next and fast.next.next:
    fast = fast.next.next
    slow = slow.next
```
结果:
```
a -> b -> c -> d -> e
          S         F
```
```
a -> b -> c -> d -> e -> f
          S         F
```
两种结果的区别就在于链表长度为偶数时, 慢指针的停留位置.
