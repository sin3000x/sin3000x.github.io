`abcabcbb` 返回 3, 因为任意长度为 4 的子串都有重复字符.
```py
from collections import defaultdict

def longest_substring(s):
    result = 0
    occurred = defaultdict(int)
    left = 0
    
    for right in range(len(s)):
        occurred[s[right]] += 1

        while occurred[s[right]] > 1:
            occurred[s[left]] -= 1
            left += 1

        result = max(result, right - left + 1)
    return result
```
